ABOUT FakeAPI

Sometimes when developing frontend app a developer needs quickly build a POC via calling some RestAPI to get data and display it in one way or another. Usually we spend some time to build some local server or something else which supports with data. FakeAPI is designed to save a developer some time. it takes care for "fake" RestAPI part which allows a developer to focus on the code. At the moment we support following APIs: person and location.
In order to use one of the APIs, simply fire a HTTP GET call to one of the given apis, like so: 
curl https://fakeapi.cc/person
or 
curl https://fakeapi.cc/location
I hope you'll find the service useful.
